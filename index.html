Projects

  • Mixed Messages

    In this project the focus was to have a random message happen anytime the user performed an action. This message consisted of three messages that would have random parts and it would be different everytime they did another action.

  • Credit Card Checker

    For this assigned product it takes a credit card number, not a real one, and returns if it is valid by using the Luhn Algorithm. It was also adapted to take in a batch of credit card numbers and return which ones were not valid as well as the credit card company they are tied to.

  • Mysterious Organism

    The purpose of this project is to first create a dna strand for an organism, either one individually or by using an object factory creating many at the same time, and then performing tasks on these strands. These tasks include comparing two strands for same bases, mutating the strand, testing how likely it will survive, and printing which organisms will survive out of a batch of them.
